## Recent Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for 10 new languages including Hindi, Arabic, and Portuguese, enabling transcription and translation for broader global teams.
- Introduced real-time summary generation during meetings, available for Zoom, Google Meet, and Teams.

## Improvements

- Enhanced transcription accuracy by 15% for noisy environments using advanced noise cancellation.
- Faster dashboard loading with optimized data fetching.

## Bug Fixes

- Fixed occasional sync issues between in-person and hybrid meeting recordings.
- Resolved translation drift in long sessions over 2 hours.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["bugfix", "security"]}>

## New Features

- Webhook support for custom integrations: Send summaries to your `https://your-webhook-url.com/webhook` endpoint.

<CodeGroup tabs="JavaScript,Python">
```javascript
const webhookData = {
  meetingId: 'mtg_123',
  summary: 'Key actions: Follow up on Q3 goals.',
  `YOUR_WEBHOOK_URL`: 'https://your-webhook-url.com/webhook'
};
```
```python
webhook_data = {
    "meeting_id": "mtg_123",
    "summary": "Key actions: Follow up on Q3 goals.",
    "webhook_url": "https://your-webhook-url.com/webhook"
}
```
</CodeGroup>

## Security Fixes

- Patched potential API token exposure in logs.

## Bug Fixes

- Corrected summary export failures for meetings with special characters.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core launch: Automatic transcription, translation, and action item extraction for all meeting types.
- Dashboard integration at `https://dashboard.example.com` for managing recordings.

## Breaking Changes

- Updated API endpoint from `/v0/transcribe` to `/v1/transcribe`. Update your calls:

</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Use `Bearer YOUR_API_KEY`.
</ParamField>

## Bug Fixes

- Initial stability improvements for hybrid meetings.

</Update>
