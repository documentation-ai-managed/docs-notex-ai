## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus" title-type="p">
    Create a free account at [https://dashboard.example.com/signup](https://dashboard.example.com/signup). No credit card required.
  </Step>

  <Step title="Install Extension" icon="download" title-type="p">
    Add the NoteX AI browser extension from the Chrome Web Store or download the desktop app.
  </Step>

  <Step title="Join a Meeting" icon="video" title-type="p">
    Start or join any supported meeting. NoteX AI activates automatically and begins capturing.
  </Step>

  <Step title="Review Output" icon="file-text" title-type="p">
    Access transcripts, translations, and summaries in your dashboard at [https://dashboard.example.com](https://dashboard.example.com).
  </Step>
</Steps>

For API integrations, use this example to fetch a meeting summary:

<CodeGroup show-lines="true" tabs={[]}>
  ```javascript
  const response = await fetch('https://api.example.com/v1/meetings/{MEETING_ID}/summary', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`
    }
  });
  const summary = await response.json();
  console.log(summary.actions);
  ```

  ```python
  import requests

  response = requests.get(
    'https://api.example.com/v1/meetings/{MEETING_ID}/summary',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  summary = response.json()
  print(summary['actions'])
  ```
</CodeGroup>
