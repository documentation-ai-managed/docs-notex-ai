## Overview

NoteX AI handles your meetings seamlessly, capturing audio from online calls, in-person discussions, or hybrid setups. You get real-time transcription, automatic translation across 50+ languages, and AI-powered summaries that highlight key decisions and action items. Save hours weekly by turning raw conversations into actionable insights without manual effort.

<Callout kind="info">
NoteX AI integrates with your calendar and meeting platforms, starting automatically when you join a call.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create a free account at https://dashboard.example.com/signup. No credit card required.
  </Step>
  <Step title="Install Extension" icon="download">
    Add the NoteX AI browser extension from the Chrome Web Store or download the desktop app.
  </Step>
  <Step title="Join a Meeting" icon="video">
    Start or join any supported meeting. NoteX AI activates automatically and begins capturing.
  </Step>
  <Step title="Review Output" icon="file-text">
    Access transcripts, translations, and summaries in your dashboard at https://dashboard.example.com.
  </Step>
</Steps>

For API integrations, use this example to fetch a meeting summary:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/meetings/{MEETING_ID}/summary', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`
    }
  });
  const summary = await response.json();
  console.log(summary.actions);
  ```
  ```python
  import requests

  response = requests.get(
    'https://api.example.com/v1/meetings/{MEETING_ID}/summary',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  summary = response.json()
  print(summary['actions'])
  ```
</CodeGroup>
