## **Pro-Tips for Better AI Results**

- **Specify the Format:** Do you want bullet points or a multi-paragraph essay? Be explicit.

- **Assign a Role:** Start with "Act as a senior financial analyst..." or "Act as a project secretary..." to set the right perspective.

- **Target Language:** Even if you've selected a default language, you can add specific nuances: "Write in Vietnamese using technical economic terminology."

## **Additional Instructions FAQ**

### What if I leave this blank?

NoteX will use a default algorithm to generate a general summary covering all the main points of the file.

### How long should my instructions be?

Keep it concise—ideally 2 to 3 sentences. This helps the AI stay focused on your core requirements.

## **FAQ – Frequently Asked Questions**

### Q: Can I export a PDF that includes the Mindmap?

A: Currently, the Export feature is optimized for text (Summary and Transcript). For Mindmaps, we recommend taking a screenshot for the highest image quality.

### Q: Why don't I see the "Download Audio" button?

A: This button only appears for notes created via "Live Recording" or "Audio Upload." Notes generated from YouTube links or Web links do not support audio downloads.

### Q: Does the Search feature look for keywords inside the summary text?

A: At this time, Search prioritizes titles and file types to ensure the fastest response time.
